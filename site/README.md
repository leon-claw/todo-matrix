# Todo Matrix Site

This directory contains the standalone homepage for `https://todo-matrix.jianghong.site/`.

It is intentionally separate from the main Todo Matrix app, which is deployed under `/app/`.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

Upload this directory's build output to the root homepage static directory, not to the main app's `/app/` directory.
